## Installation 

1. **Clone the repository**:
   
    git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
    cd YOUR_REPO_NAME
   

2. **Create a virtual environment**:
   
    python -m venv venv
    venv\Scripts\activate
   

3. **Install dependencies**:
    pip install -r requirements.txt
